## Which LemonSlice product should I use?

There are 3 ways to use LemonSlice's interactive avatar models: (1) **Widget** - low code and fastest, (2) **Hosted Pipeline** - customize the UI, and (3) **Self-Managed Pipeline** - customize everything.
