## What is the LemonSlice Widget?

A fully-managed conversational AI service and UI component. Use it when you want the fastest path to adding an interactive avatar to your site.

Create a video agent in the LemonSlice web app. Customize the appearance, voice, personality, and knowledge base of your agent. Finally, copy the embed code and paste it on your site.


## Get started now

Add the widget to any webpage with a single HTML tag in your header.

```html
<lemon-slice-widget agent-id="agent_id"></lemon-slice-widget>
<script type="module" src="https://unpkg.com/@lemonsliceai/lemon-slice-widget"></script>
```
